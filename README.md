# Humanitarian Health & Nutrition Analytics System (HHNAS)

> A real-world end-to-end Python Data Analytics project built to simulate the daily work of a Data Analyst in the humanitarian sector.

---

# 📖 Project Overview

The **Humanitarian Health & Nutrition Analytics System (HHNAS)** is a portfolio project that simulates a real analytics workflow used by humanitarian organizations.

Instead of solving isolated coding exercises, this project follows the complete lifecycle of a professional data analytics project:

- Environment setup
- Project organization
- Version control using Git & GitHub
- Data collection
- Data cleaning
- Data transformation
- Exploratory Data Analysis (EDA)
- Statistical analysis
- Data visualization
- Business reporting

The project is developed step by step over seven days, following professional software development practices.

---

# 🎯 Project Goals

- Build a production-quality Python project.

- Apply Python to realistic humanitarian datasets.
- Learn data cleaning and preparation.
- Perform exploratory data analysis.

- Generate visualizations and reports.
