# Humanitarian Health & Nutrition Information System (HHNIS)

## Overview

This document describes the relational database design for the Humanitarian Health & Nutrition Analytics Platform.

---

# Database Objectives

- Store operational health data.
- Store nutrition program data.
- Support reporting and KPIs.
- Enable SQL analysis.
- Support Power BI dashboards.
- Simulate real humanitarian information systems.

---
